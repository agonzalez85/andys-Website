Install Python Environment
  'python3 -m venv pyenv'
  'source pyvenv/bin/activate'

Install Flask
  'pip install flask'

Clone Repo
  'git clone git@github.com:arelyyrosendes/Andys-Website.git'

Run Website Locally!
  'python3 app.py'

Deploy to Vercel
  1. Install Vercel CLI: `npm i -g vercel`
  2. Login: `vercel login`
  3. From the repo root run: `vercel`
  4. For subsequent deploys: `vercel --prod`
